# Проект Chat Application

Этот проект представляет собой веб-приложение для чата, разработанное с использованием Streamlit. Приложение позволяет пользователям общаться в режиме реального времени и сохраняет историю чатов в базе данных PostgreSQL.

## Описание проекта

Проект состоит из следующих основных компонентов:

1. **app.py**: Главный файл приложения, который инициализирует базу данных и рендерит пользовательский интерфейс для настроек и чата.
2. **.env**: Файл с переменными окружения, необходимыми для подключения к базе данных и настройки модели.
3. **docker-compose.yml**: Файл для настройки и запуска контейнеров Docker для веб-приложения и базы данных PostgreSQL.
4. **Dockerfile**: Файл для создания Docker-образа приложения.

## Установка и запуск

### Локальный запуск

1. Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```

2. Создайте файл `.env` в корневой директории проекта и добавьте в него следующие переменные окружения:
    ```env
    MODEL_NAME="llama-3-8b-instruct-8k"
    MODEL_PROVIDER="openai"
    BASE_URL="https://llama3gpu.neuraldeep.tech/v1"

    DB_NAME="chat_history"
    DB_USER="postgres"
    DB_PASSWORD="postgres"
    DB_HOST="localhost"
    DB_PORT="5432"
    ```

3. Запустите приложение:
    ```bash
    streamlit run app.py
    ```

### Запуск с использованием Docker

**На данный момент Docker не работает.** 

Для запуска проекта с использованием Docker, выполните следующие шаги:

1. Создайте файл `.env` в корневой директории проекта и добавьте в него переменные окружения, как указано выше.
2. Запустите команду:
    ```bash
    docker-compose up --build
    ```

## Использование

После запуска приложения откройте браузер и перейдите по адресу `http://localhost:8501`. Вы увидите интерфейс для настройки и чата. Введите свои настройки и начните общение.

## Примечания

- Убедитесь, что у вас установлен Docker и Docker Compose.
- Проверьте правильность переменных окружения в файле `.env`.
